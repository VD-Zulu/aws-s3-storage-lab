# AWS S3 Storage Lab

## Objective
This lab demonstrates:
- Amazon S3
- Object storage
- Storage classes
- Cloud storage security

---

## Tasks Completed
- Created S3 bucket
- Uploaded objects
- Tested storage classes
- Configured permissions
- Analyzed storage risks

---

## Key Learnings
- S3 uses object storage
- Public storage creates risk
- Storage classes optimize cost
- Governance is critical in cloud storage

---

## Security Reflection
Misconfigured cloud storage can expose sensitive enterprise data publicly.
